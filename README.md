# git-practice-base

A git repositorey to practice git controls.

## Basic

    git clone https://github.com/othree/git-practice-base.git

Edit `README.md`. Then commit.

### Edit commit message

    git commit --amend

or by `reset`

    git reset --soft HEAD^

zsh:

    git reset --soft HEAD\^

git reset can used to edit entire commit.
